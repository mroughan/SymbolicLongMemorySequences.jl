# Uniform Marginal Validation

Generated by `validation/marginal_control.jl`.

- Target marginal: uniform categorical distribution.
- Alphabet size: `k = 8`.
- Sequence length: `n = 100000`.
- Trimmed length: `80000` after dropping the first and last 10%.
- Replicates per method: `20`.

The chi-squared p-values use the iid multinomial reference distribution.
They are useful frequency diagnostics, but they are not exact tests for
dependent LRD sequences. LRD can slow convergence of empirical
marginals, so this reference is often conservative.

The `effective_n_*`, `chi2_ess_*`, `pvalue_ess_*`, and
`reject_ess_005` columns apply an approximate effective-sample-size
correction. The correction estimates an integrated autocorrelation time
for each centered one-hot symbol indicator, uses the smallest symbol
effective sample size, and scales the chi-squared statistic by
`effective_n / trimmed_n`. It is a dependence-aware diagnostic, not an
exact categorical LRD test.

The summary CSV reports both full-sequence marginal errors and
trimmed-window marginal errors. The chi-squared diagnostics and
histogram data use the trimmed window.

More formal marginal tests should use a calibrated dependence-aware
null, such as block/subsampling calibration or a parametric Monte Carlo
envelope generated from the same configured method.
